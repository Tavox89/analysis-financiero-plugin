# Finanzas ASD

Finanzas ASD es la suite financiera modular de ASD Labs para centralizar documentos, saldos, cobros, pagos, cuotas, prestamos y reportes en una sola base.

## Principios

- lenguaje claro y formal
- gestion primero, reporte despues
- sincronizacion real con otros plugins
- un solo plugin activo
- arquitectura modular por dentro
- tablas propias para el core financiero

## Lenguaje del sistema

- Documento
- Perfil
- Contacto externo
- Cuenta
- Compra
- Mercancia por recibir
- Cobro
- Pago
- Cuota
- Prestamo
- Reporte
- Integracion
